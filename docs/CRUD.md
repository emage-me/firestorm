# CRUD

### Create
Id is generated by firebase if not provided
```javascript
const userData = {
  id: '42',
  firstName: 'Jack'
}
const user = new User(userData)
await user.save()
```

### Read
```javascript
const user = new User.find('42')
```

### Update
```javascript
await user.update({firstName: 'luke'})

await user.set({firstName: 'luke'})

user.firstName = 'luke'
await user.save()
```

### Delete
```javascript
await user.delete()
```
